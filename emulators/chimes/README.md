# Mac sounds

Classic Macintosh startup chimes and matching system-error ("crash") sounds. The setup script picks a startup chime, installs it as `/usr/local/bin/chime.wav` (played at boot), and installs the era-matched crash sound as `/usr/local/bin/crash.wav` (played when the user hits the reset button).

## Startup chimes

| File                                | Era / Model                       |
| ----------------------------------- | --------------------------------- |
| `StartupMacI.wav`                   | Mac 128K — 1984                   |
| `StartupMacII.wav` *(default)*      | Mac II — 1987                     |
| `StartupMacLC.wav`                  | Mac LC — 1990                     |
| `StartupMacQuadra.wav`              | Quadra — 1991                     |
| `StartupMacQuadraAV.wav`            | Quadra AV — 1993                  |
| `StartupPowerMac.wav`               | Power Mac — 1994                  |
| `StartupPowerMacCard.wav`           | Power Mac (PCI card variant)      |
| `StartupTwentiethAnniversaryMac.wav` | Twentieth Anniversary Mac — 1997 |

## Crash sounds

| File                       | Era / Model       | Picked for…                              |
| -------------------------- | ----------------- | ---------------------------------------- |
| `CrashMacII.wav`           | Mac II — 1987     | Mac I, Mac II                            |
| `CrashMacIIfx.wav`         | Mac IIfx — 1990   | (unused; available)                      |
| `CrashMacLC.wav`           | Mac LC — 1990     | Mac LC                                   |
| `CrashMacQuadra.wav`       | Quadra — 1991     | Quadra                                   |
| `CrashMacQuadraAV.wav`     | Quadra AV — 1993  | Quadra AV, Power Mac (×2), Twentieth Anniversary |
